GAME GEEKS

La aplicacion Game Geeks fue programada en visual community el cual utilice el lenguaje de C++ para poder instalarlo este va conectado a una base de datos local la cual podemos registrar nuevos usuarios y poder iniciar como usuario.

Menu

Nuestro menu consta de 8 opciones para que el usuario acceda a ellas y son las siguientas:

  • Noticias Geeks
  • Noticias de videojuegos
  • peliculas y trailers
  • Ultimos lanzamientos
  • Foros Geeks
  • Productos Geeks
  • Redes Sociales
  • Ayuda

La aplicacion esta dirigida a fans de peliculas y videojuegos la cual nos permite estar actualizados sobre nuevas peliculas, series y videjuegos, anexo imagenes de la aplicacion funcionando.
